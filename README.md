# UK Railway Data Analysis

A comprehensive data analytics project analysing UK railway ticket transactions to uncover insights on journey performance, revenue patterns, and customer behavior.
## Project Overview
This project demonstrates end-to-end data analytics skills using Python, PySpark, and SQL on Databricks. The analysis covers 31,653 railway transactions with 18 features including ticket details, journey status, delays, and refund requests.
### Key Business Questions Answered
- Journey Performance: What is the on-time performance rate and what causes delays?
- Revenue Analysis: Which ticket types generate the most revenue?
- Station Performance: Which stations have the highest delay rates?
- Customer Behavior: How do customers purchase tickets and what affects refund requests?
- Time Patterns: What are the peak travel hours and days?
